# CLAUDE.md — проект «Трансфер Север»

## Что это
Демо-лендинг пассажирских перевозок по маршруту Красноярск — Северо-Енисейский.
Цель версии — показать заказчику (ИП Ежкова Е.М.), что попали в тематику и стиль.
Не финальный продукт: реквизиты реальные, но MAX-бот и тексты политик — заглушки.

## Стек
- Чистый HTML + CSS + JavaScript. Без фреймворков и сборщиков.
- Принцип mobile-first. Главная цель — телефон.
- Хостинг демо — временная ссылка (Netlify / Vercel / GitHub Pages).

## Порядок работ
Сайт собираем **по частям, не весь сразу**. Этапы и их статусы — в `ROADMAP.md`.
После каждого этапа — показ Коляну и согласование, потом следующий этап.
Начинать с актуального незакрытого этапа в ROADMAP.

sitemap.xml/schema.org. Полный список — раздел 20 в `tz-FINAL.md`.

## Запуск сайта

Сборки нет — чистый HTML/CSS/JS, открывается напрямую.

**Вариант 1 — Python (встроен в систему, не требует Node):**
```bash
python -m http.server 5500
```
Сайт: http://localhost:5500

**Вариант 2 — live-server (авто-перезагрузка при изменениях, нужен Node):**
```bash
npx -y live-server --port=5501 --no-browser
```
Сайт: http://localhost:5501

Конфиг для `/run` скилла — `.claude/launch.json`.

## Правила работы
- Перед кодом обсуждаем (см. tz-FINAL.md). Не торопиться.
- Комментарии в коде — на русском. Имена переменных/функций — на английском.
- Заказчик хочет заявки в MAX. Без бота это невозможно; бот требует верификации
  реального ИП — это задача боевой версии (раздел 22 в `tz-FINAL.md`).
